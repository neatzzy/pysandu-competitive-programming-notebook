\subsection{Algorithms/Concepts}

\begin{itemize}
    \item \textbf{Convex Hull (Monotone Chain or Graham Scan): }Finds the smallest convex polygon containging all points
    \item \textbf{Line Segment Intersection: }Using cross products to check if two segments intersect
    \item \textbf{Point in Polygon: }Using ray casting (winding number) algorithm for O(n) complexity or BSP Tree for O(log n) complexity
    \item \textbf{Angle of vector: }Use \texttt{atan2(y,x)} to find the angle of a vector.
\end{itemize}
