\subsection{System optimization}

For faster IO, we can use the following code:
\begin{lstlisting}[language=C++]
ios::sync_with_stdio(false);
cin.tie(nullptr);
\end{lstlisting}
The first line disables synchronization between C++ and C streams; the second unties \texttt{cin} from \texttt{cout} so flushing doesn't happen before each read. Together they can speed up IO by 10$\times$ or more. \texttt{cout.tie(nullptr)} is redundant here and can be omitted.

\subsection{Pragmas GCC Optimization}
Write these directives on the \textbf{first line} of your code (before any include) to force the compilator to perform more aggressive optimizations.:

\begin{lstlisting}[language=C++]
#pragma GCC optimize("O3,unroll-loops")
#pragma GCC target("avx2,bmi,bmi2,lzcnt,popcnt")
\end{lstlisting}
