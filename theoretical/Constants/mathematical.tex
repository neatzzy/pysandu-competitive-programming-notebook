\subsection{Mathematical}

\begin{itemize}
    \item \textbf{Pi ($\pi$)}: Usually defined as acos(-1.0), approximately equal to 3.14159. In C++ 20, it is available as \texttt{std::numbers::pi}.
    \item \textbf{Euler ($e$)}: Usually defined as the base of the natural logarithm, approximately equal to 2.71828. In C++ 20, it is available as \texttt{std::numbers::e}.
    \item \textbf{Golden Ratio ($\phi$)}: Usually defined as (1 + sqrt(5)) / 2, approximately equal to 1.61803. In C++ 20, it is available as \texttt{std::numbers::phi}.
\end{itemize}
