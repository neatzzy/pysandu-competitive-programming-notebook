\subsection{System of Linear Equations}

A system of linear equations $Ax = b$ (with $A$ an $n \times n$ matrix) has:
\begin{itemize}
    \item \textbf{Unique solution} if $\det(A) \neq 0$.
    \item \textbf{Infinitely many solutions} if $\det(A) = 0$ and the system is consistent.
    \item \textbf{No solution} if $\det(A) = 0$ and the system is inconsistent.
\end{itemize}

\subsection{Linear System Solving}

\textbf{Gaussian Elimination with Partial Pivoting} — O($n^3$). Augment to $[A \mid b]$ and apply row operations:
\begin{enumerate}
    \item For column $j$: find row $i \geq j$ with the largest $|A_{ij}|$; swap rows $i$ and $j$ (partial pivoting for numerical stability).
    \item Divide row $j$ by the pivot $A_{jj}$.
    \item Subtract appropriate multiples of row $j$ from all other rows to zero out column $j$.
\end{enumerate}
After elimination, $x$ is read directly from the last column. If a pivot is zero (or near-zero), the matrix is singular and the system has no unique solution.
