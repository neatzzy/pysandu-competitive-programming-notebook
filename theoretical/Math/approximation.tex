\subsection{Numerical Algorithms}

\begin{itemize}
    \item \textbf{Newton's Method: } Finds roots of a differentiable function iteratively:
          \[
              x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}
          \]
          Converges quadratically near the root. Also used for computing integer square roots and modular inverses. May diverge if the initial guess is far from the root or $f'(x_n) \approx 0$.

    \item \textbf{Binary Search on Real Numbers (Ternary Search): } To find the minimum/maximum of a unimodal function on $[lo, hi]$, repeatedly evaluate at $m_1 = lo + (hi-lo)/3$ and $m_2 = hi - (hi-lo)/3$ and discard the worse third. Converges in O($\log_{3/2} \frac{1}{\epsilon}$) iterations.

    \item \textbf{Simpson's Rule: } Approximates a definite integral using parabolic segments:
          \[
              \int_a^b f(x)\,dx \approx \frac{b-a}{6}\left[f(a) + 4f\!\left(\tfrac{a+b}{2}\right) + f(b)\right]
          \]
          For higher accuracy, apply adaptively: recurse when \texttt{|simpson(a,m)+simpson(m,b) - simpson(a,b)|} $> \varepsilon$.

    \item \textbf{Trapezoidal Rule: } Simpler integration approximation using trapezoids:
          \[
              \int_a^b f(x)\,dx \approx \frac{b-a}{2}[f(a) + f(b)]
          \]
          Less accurate than Simpson's for the same step size; use Simpson's in competitive programming.
\end{itemize}
